Support

Une question, un problème avec l'application Cyclicare ou votre accès ? Nous sommes là.

Contactez-nous : cyclicareoff@outlook.com
Nous répondons généralement sous 48 heures ouvrées.

Questions fréquentes

  • Je n'arrive pas à me connecter — vérifiez votre adresse email ; vous pouvez vous connecter par mot de passe ou recevoir un code de vérification par email.
  • Mon accès est verrouillé — écrivez-nous, nous réactivons votre espace.
  • Mode partenaire — votre code d'invitation se trouve dans l'onglet « Nous » de l'application. Vous pouvez le régénérer ou rompre le lien à tout moment.
  • Supprimer mon compte — directement dans l'application : Profil → « Supprimer mon compte » (délai d'annulation de 30 jours).
  • Exporter mes données — Profil → « Exporter mes données » (PDF complet).
